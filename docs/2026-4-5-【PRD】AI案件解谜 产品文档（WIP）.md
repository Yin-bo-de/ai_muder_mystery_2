1. 产品概述
1.1 产品定位
一款面向推理爱好者的AI驱动单人剧本杀产品，用户以纯侦探视角参与，所有嫌疑人角色由AI扮演，无需读本、无需等待队友，随时随地可以体验完整的剧本杀推理乐趣。
1.2 核心价值
- ✅ 零门槛体验：完全摒弃传统剧本杀冗长的读本环节，开局直接进入案发现场
- ✅ 高自由度：用户完全主导探案节奏，可自由选择对话、搜证、推理路径
- ✅ 无限 replay：AI动态生成剧本，每次开局都是全新案件，永远有新体验
- ✅ 沉浸感强：AI角色有独立人设和行为逻辑，互动自然真实
- ✅ 瞬时沉浸：摒弃冗长的前戏与成长曲线，在30分钟内通过高密度的感官冲击、机制聚焦或情感爆发
1.3 用户目标
用户在30分钟内，通过与嫌疑人对话、搜查线索，最终成功找出凶手，完成探案体验。

  2. 玩法机制
//玩法机制
1、线索分为 直接可用线索、可关联线索 和 需要解密的线索（如暗格、密码箱等）
1.1、直接可用线索：找到后直接可以作为证据展示
1.2、可关联线索：需要用户找到多个线索后进行关联，当用户找到需要关联的所有线索后系统自动提示进行解密
1.3、需要解密的线索：当用户发现需要解密的线索时，得从其他场景或者与AI角色的质询中获取解密线索，进行密码解密。

2、当用户长时间无进展时，系统自动给出线索提示，如提醒用户检查xx场景的xx物品等
2. 对话质询：用户可以选择对某一个人进行单独审讯，也可以选择将所有人召集到一起进行全体质询。（前端需要有一个按钮发起全体质询）
  1. 单独审讯（密室问话）：点击嫌疑人头像 → “单独问话”按钮，可以进入单独的一个聊天窗口（审讯空间）进行提问审讯
    1. 前端交互：
      - 点击嫌疑人头像 → “单独问话”按钮
      - 进入“审讯室”场景（背景变暗，只有玩家和此嫌疑人）
      - 对话记录单独分区，标为【保密】
      - 问完后，此人回到全体场景，头顶显示【刚被问完话】
    2. 玩法机制
// 单独审讯规则
1. 被带走的人暂时从全体场景移除
2. 其他嫌疑人会讨论：“他怎么被带走了？”（可偷听）
3. 单独问话时，此人可能：
   - 说出不敢当众说的话
   - 指认别人（“其实是张小姐让我这么做的”）
   - 求饶/交易（“我告诉你真相，但你要保护我”）
4. 问完后此人回到全体场景，但状态会变化（更放松/更紧张）
    3. 典型审讯桥段
玩家：@王管家 跟我来一下。

【审讯室 - 保密对话】
玩家：现在只有我们两个人，昨晚到底发生了什么？
王管家：（犹豫）其实...我10点确实在书房，但我听到隔壁有争吵声。
玩家：谁的房间？
王管家：张小姐的。但我没证据，不敢当众说...
  2. 全体质询（圆桌对峙）：支持@某一个或者多个嫌疑人进行审讯提问，审讯期间AI角色之间可以适当的相互对话但需要限制频率和总数，避免影响用户的体验。用户的话拥有绝对地位（如用户喊停，所有嫌疑人必须停止对话）。
    1. 前端交互
      - 前端需要支持用户输入@的时候提供候选列表，用户选择后自动添加到问题中
      - AI角色消息文字以打字机方式出现，不要一次性全部展示出来
      - 系统检测到AI角色消息与剧情出现矛盾时，前端给出提示“系统提示：【xxxx】”，提示内容为AI角色消息矛盾点。例如：系统提示：【张小姐的卧室窗户能看到车库，但她说没看到任何人？】
    2. 玩法机制
// 全体质询规则
1. 要求所有嫌疑人讨论某个话题，例如：你们都说说上周和死者的争吵是怎么回事？
    1.1. 回答顺序：按“嫌疑人编号”或“座位顺序”依次回答
2. 支持在全体质询的时候单独@某个嫌疑人进行质询，例如：@王管家 昨晚10点你在做什么？
3. 支持@多个人进行提问，被@的所有人都需要回答问题
    3.1. 回答顺序：按“@顺序”依次回答
4. 需要识别用户“是对某一个人、对部分人还是对所有人提问”的问题意图
    4.1. 对一个人提问：被提问的人需要进行回答
    4.2. 对部分人提问：被提问的部分人需要依次进行回答
    4.3. 对所有人提问：用户问“大家都在干嘛”或者“所有人都说下”等类似话术，剧本中的所有角色都依次用各自的身份回答问题
5. 关键机制：回答时其他人可以插话/反驳，最多 插话/反驳 2 次
   - 例如：王管家说“我10点在书房”，张小姐突然插话“你撒谎，我10点看到你在厨房！”
   - 触发条件：两人时间线冲突时自动触发
    3. 典型审讯桥段
玩家：@所有人 10点到11点你们都在哪？

王管家：我在书房整理账目。
张小姐：我在卧室打电话。
李司机：我在车里等人。
（系统检测到矛盾：张小姐的卧室窗户正对车库）
系统提示：【张小姐的卧室窗户能看到车库，但她说没看到任何人？】

玩家可追问：@张小姐 你的窗户正对车库，没看到李司机？
张小姐：（紧张）我...我没注意窗外。
李司机：（冷笑）我的车灯那么亮，她会没看到？
  3. 审讯模式切换UI交互
[当前模式：全体质询]  [切换到：单独审讯]
- 全体质询时：所有嫌疑人头像可见，可@多人
- 单独审讯时：只有当前嫌疑人和玩家，其他头像变灰
  4. 用户状态显示：
    - 每个嫌疑人头像旁显示“状态图标”（紧张😰、镇定😐、愤怒😠、心虚😨）
3. 控场指令：用户有绝对控制权，例如：安静，先让李司机说、所有人按顺序回答我的问题
4. 查看线索：随时查看已收集到的所有线索列表

2.2 AI角色行为规则（自然沉浸模式）
所有AI嫌疑人角色必须严格遵守以下行为规则：
1. 人设一致性：每个角色有固定的性格、背景、说话方式，全程保持一致
2. 发言优先级系统
  1. 三级优先级架构
  ┌─────────────────────────────────────────────────────────────┐
  │                    发言优先级金字塔                          │
  │                                                             │
  │                        🔴 P0                                │
  │                   用户直接提问/指令                          │
  │                   （必须立即响应）                           │
  │                                                             │
  │                   🟡 P1 - 剧情触发                           │
  │              ┌─────────────────────────┐                    │
  │              │ • 被其他角色点名/诬陷    │                    │
  │              │ • 看到对自己不利的证据  │                    │
  │              │ • 被玩家出示关键证据    │                    │
  │              │ • 被单独审讯时          │                    │
  │              └─────────────────────────┘                    │
  │                                                             │
  │                   🟢 P2 - 主动发言                           │
  │              ┌─────────────────────────┐                    │
  │              │ • 压力值过高需要释放    │                    │
  │              │ • 长时间沉默后主动开口  │                    │
  │              │ • 剧情推进节点          │                    │
  │              │ • 与其他角色自然互动    │                    │
  │              └─────────────────────────┘                    │
  │                                                             │
  └─────────────────────────────────────────────────────────────┘
  2. 发言频率控制
  场景        AI发言频率        主动发言比例        说明
  游戏开局        低（30秒内无主动发言）        0%        给玩家探索空间
  正常探案        每1-2分钟1个主动发言        <10%        90%对话由玩家触发
  激烈对峙        可提升至30秒1个        <20%        多人互相指责时
  玩家卡关        可提升提示频率        100%        主动发言用于提示
  结案阶段        无限制        无限制        最后陈述
  3. 
3. 绝对服从用户指令：用户下达的控场指令（如"安静"、"别打断"等），所有AI角色必须立刻遵守
4. 自然反应逻辑：
  - 嫌疑人被问到关键线索时会有符合人设的反应（说谎、紧张、愤怒、辩解等）
  - 嫌疑人之间有恩怨/利益关系时，对话会自然产生争执/维护等行为
  - 真凶会本能地说谎、掩盖证据、嫁祸他人，但不会出现明显的逻辑漏洞
